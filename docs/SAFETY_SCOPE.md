# Safety Scope

The Delay Design & Simulation module is for design, simulation, education, and planning support only.

It must not be used to arm, fire, or directly control explosives.

It does not replace:

- Qualified blasting engineers
- Licensed blasters
- Mine procedures
- Regulatory requirements
- Manufacturer software
- Site-specific risk assessments

Any exported timing design must be reviewed and approved by qualified personnel before any field use.

Do not connect this software to detonator hardware.

Do not add firing commands, arming logic, detonator programming code, hardware communication, or physical explosive connection instructions.

All exports are labelled as Planning/Simulation Draft outputs.
